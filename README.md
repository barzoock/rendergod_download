## Key Features

### UnCrash - Automatic Render Recovery
The core of RenderGod. An intelligent recovery system that ensures your Blender renders always finish successfully.

- **Crash Recovery** — Detects Blender crashes instantly and resumes from the exact frame where it stopped
- **Freeze Detection** — Catches stuck frames and restarts automatically to prevent endless hangs
- **Survives System Reboots** — Resumes rendering after PC crashes or power loss, bypassing the Windows login screen via a secure encrypted session
- Smart retry limits, Telegram alerts, fully customizable

### Telegram Bot - Remote Control From Anywhere
Not just notifications — a full **command center in your pocket**.

- **Live Previews** — Video and image previews sent directly to your phone
- **Remote Management** — Pause, stop, or prioritize renders via chat commands
- **Real-Time Updates** — Know the second a render is done

### Advanced Render Queue
Manage multiple render tasks and media conversions simultaneously.

- Real-time status tracking with remaining time estimates
- Bulk processing with custom presets
- The ultimate dashboard for your Blender production pipeline

### Batch Generator
Auto-generate queue items for all scenes, cameras, and view layers with one click. Stop wasting time adding items manually.

- Multi-scene support
- Auto-camera detection
- Any combination of scenes, cameras, and view layers

### Live Preview
Instant visual feedback for every frame — no more "render and pray".

- Real-time playback during rendering
- Scroll-wheel zoom
- Grid view for output nodes and layers
- Full EXR support including multilayer

### Boost Performance
Maximize your hardware with integrated Blender and system-level optimizations.

- System-level and GPU-level optimizations
- Scene optimization toggles
- GPU accelerated (NVIDIA supported)
- **Tested: 18.11% faster renders** on RTX 4090 (Classroom scene, 32 samples, OptiX denoiser)

### Image Sequence to Video Conversion
Convert image sequences to video or extract frames from videos — a lightning-fast workflow for final delivery.

- One-click export
- Professional media encoders (including ProRes Alpha)

### Smart Export
Automate your entire export folder structure across all scenes, cameras, and view layers.

- Dynamic folder paths
- Auto-versioning system
- Native File Output Node support for complex compositor setups

### Actions When Done
Execute custom actions automatically after rendering completes.

- Auto sleep / shutdown
- Auto video from image sequence
- Custom post-render actions

### Native Blender Addon
One click inside Blender to start rendering — a seamless bridge between your scene and RenderGod.

- Available in Blender's sidebar
- One-click connection

### Auto Bake Physics
All physics simulations (rigid bodies, cloth, fluids, smoke) are automatically baked before rendering begins. No mid-render stalls, no broken simulations.
