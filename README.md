### Features

- 🎬 Blender render management with automatic crash detection and recovery
- 🔄 Automatic updates - the app will notify you when new versions are available
- 📹 Image sequence to video conversion tools
- 📱 Telegram notifications for remote monitoring
- ⚙️ Customizable render settings and presets

### System Requirements

- **OS:** Windows 10/11 (64-bit)
- **Blender:** Version 2.8 or higher
- **RAM:** 4GB minimum (8GB+ recommended)
- **Storage:** 100MB for application

### Auto-Updates

RenderGod includes automatic update checking. When a new version is available, you'll receive a notification within the app with the option to download and install the update.
